Тест кейс №1
Название тест кейса: Поиск по услуги мастерской по названию
Шаги тест кейса:
Шаг тест кейс №1: Открыть приложение, нажать кнопку "Войти"
Шаг тест кейса №2: Ввести в поле для ввода текст услуги, например: "Замена актуатора сцепления"
Ожидаемый результат тест кейса: Выводится услуга сервиса с набранным названием

Шаги тест кейса:
Шаг тест кейс №1: Открыть приложение, нажать кнопку "Войти"
Шаг тест кейса №2: В графе фильтр выбрать любой из заданных параметров, например: "Title"
Ожидаемый результат тест кейса: Вывод услуг по заданному параметру(фильтру)

Тест кейс №3
Название тест кейса: Вход в приложение автомастерской с правами администратора
Шаги тест кейса:
Шаг тест кейс №1: Открыть приложение, нажать кнопку "Войти"
Шаг тест кейса №2: Нажать кнопку "Администратор"
Шаг тест кейса №3: В поле для ввода ввести пароль "0000"
Шаг тест кейса №4: Нажать войти
Ожидаемый результат тест кейса: Откроется список с выбором услуг и возможностью редактирования, а так же некоторыми другими функциями
